## 1. Product Overview

### Vision
Transform the existing Flutter mobile CGPA/GPA Calculator into a modern, responsive web application that helps students track and calculate their academic performance across semesters. Built with Next.js 14+ and Supabase for a seamless, full-stack experience.

### Target Users
- **Students** at all education levels (high school, college, university)
- **Academic advisors** helping students track performance
- **Parents** monitoring their children's academic progress

### Core Value Proposition
A professional, easy-to-use web platform for calculating weighted GPA (Grade Point Average) and CGPA (Cumulative Grade Point Average) with beautiful visualizations, real-time calculations, and secure cloud storage.

### 6.2 Row Level Security (RLS) Policies

**Enable RLS on all tables and create policies:**

```sql
-- Users can only read/write their own settings
CREATE POLICY "Users can manage own settings"
ON user_settings
FOR ALL
USING (auth.uid() = user_id);

-- Users can only read/write their own semesters
CREATE POLICY "Users can manage own semesters"
ON semesters
FOR ALL
USING (auth.uid() = user_id);

-- Users can only read/write subjects in their semesters
CREATE POLICY "Users can manage own subjects"
ON subjects
FOR ALL
USING (
  semester_id IN (
    SELECT id FROM semesters WHERE user_id = auth.uid()
  )
);

-- Users can only read/write their own grade configs
CREATE POLICY "Users can manage own grades"
ON grade_configs
FOR ALL
USING (auth.uid() = user_id);
```

## 9. Calculation Logic

### GPA Calculation Function
```typescript
function calculateSemesterGPA(subjects: Subject[]): number {
  if (subjects.length === 0) return 0;
  
  let totalGradePoints = 0;
  let totalCredits = 0;
  
  for (const subject of subjects) {
    totalGradePoints += subject.gradePoints * subject.credits;
    totalCredits += subject.credits;
  }
  
  if (totalCredits === 0) return 0;
  
  const gpa = totalGradePoints / totalCredits;
  return Math.round(gpa * 100) / 100; // 2 decimal places
}
```

### CGPA Calculation Function
```typescript
function calculateCGPA(semesters: Semester[]): number {
  if (semesters.length === 0) return 0;
  
  let totalGradePoints = 0;
  let totalCredits = 0;
  
  for (const semester of semesters) {
    for (const subject of semester.subjects) {
      totalGradePoints += subject.gradePoints * subject.credits;
      totalCredits += subject.credits;
    }
  }
  
  if (totalCredits === 0) return 0;
  
  const cgpa = totalGradePoints / totalCredits;
  return Math.round(cgpa * 100) / 100; // 2 decimal places
}
```

    return {

      color: 'yellow',
      description: 'Satisfactory academic performance'
    };
  } else {

      level: 'Below Average',
      color: 'red',
      description: 'Needs improvement'
    };
  }
}
